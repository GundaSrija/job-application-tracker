# Job Application Tracker

A Python-based console application to track job applications efficiently.

## Features
- Add job applications
- View all applications
- Update application status (Applied, Interview, Rejected, Offer)
- Delete applications
- JSON-based data persistence

## Technologies Used
- Python
- JSON
- File handling

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run the program:
   ```bash
   python main.py
**Project Description:
This project helps users manage and track job applications through a simple menu-driven interface.
All application data is stored persistently using a JSON file, allowing users to retain data between program executions.
